# 🛒 Amazon Clone (HTML, CSS, JavaScript)

This project is a fully responsive Amazon Clone web application developed using HTML, CSS, and JavaScript. It replicates the core design and basic functionality of the Amazon e-commerce platform, focusing on front-end development and user experience.

## 🚀 Features

- 🏠 Amazon-like homepage with banner and product sections
- 🛍️ Add to Cart functionality
- ➕ Increase/Decrease product quantity
- 💰 Dynamic total price calculation
- 🧾 Cart summary and item listing
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Clean UI inspired by Amazon

## 🛠️ Tech Stack

- HTML5 – Structure of the website
- CSS3 – Styling and layout (Flexbox/Grid)
- JavaScript (ES6) – Functionality and interactivity

## 📂 Project Structure

- `index.html` – Main homepage
- `style.css` – Styling file
- `script.js` – JavaScript logic
- `images/` – Product and UI images

## ⚙️ Functionality

This project simulates a basic e-commerce workflow:
- Users can browse products displayed on the homepage
- Add items to the cart
- Modify item quantities
- View updated total price dynamically

## 🎯 Objective

The goal of this project is to:
- Practice core web development (HTML, CSS, JavaScript)
- Build a real-world UI inspired by Amazon
- Understand DOM manipulation and event handling
- Improve responsive design skills
